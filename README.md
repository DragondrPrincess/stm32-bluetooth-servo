**#STM32 + Android 蓝牙舵机控制项目**

一个通过蓝牙（HC-05）由安卓应用控制舵机的项目。

**##硬件配置**

\-主控：STM32 系列单片机

\-蓝牙模块：HC-05蓝牙串口模块

\-执行器：舵机

**##功能特性**

安卓端滑块实时控制舵机旋转角度（0°-180°）

基于蓝牙串口 HC-05 的稳定通信协议

舵机当前角度可显示到手机界面

**##项目结构**
